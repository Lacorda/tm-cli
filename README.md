# tm-lite
一个用来管理开发模板的脚手架

## 安装

```
npm install tm-lite -g
```

## 使用
```
tm -h
```

```
Usage: tm <command>

Options:
  -V, --version  output the version number
  -h, --help     output usage information

Commands:
  add            add a new template
  delete         delete a template
  list           List the templateList
  init           init a project
```
